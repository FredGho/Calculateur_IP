Le calculateur "L'Âne de Florent" vous permet de calculer une plage d'adresses IP. Cette plage d'adresses IP vous permet de connaître le nombre d'adresses IP que vous devez réserver pour votre réseau en limitant le gâchis (c'est à dire les adresses IP que vous n'utiliserez pas). Pour que le calculateur donne votre plage d'adresses IP, vous devez renseigner une adresse IP. Vous devez également renseigner un CIDR ("Classless Inter Domain Routing", routage inter-domaine sans classe en français).
Le CIDR est un nombre entre 1 et 32 compris. Il indique le nombre de bits allumés dans le masque de sous-réseau
